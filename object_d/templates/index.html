Video Streaming Demonstration

2018 NUGU 공모전 심사용 인공지능 영상 분석 데모 서버 입니다.

(본 URL 접속시 인공지능 모델을 불러오기 위해 약 20초간의 로딩후에 상단에 영상이 나타납니다)

인공지능을 활용하여 영상 데이터에서 검출된 결과를 보여줍니다.

** 현재 제출된 play와 연동된 proxy 서버는 데모 심사를 위해, 편의상 녹화된 비디오 파일의 반복을 입력 소스로 사용 하였습니다.

** 본 영상 송출 서버에 GET을 호출하는 시점에 비디오 소스의 시작점에서 결과를 보여주므로 play의 응답과 현재 위에 보이는 영상 결과가 일치하지 않는점을 유의해 주시기 바랍니다.

** 실제 서비스시 사용자가 등록한 실시간 영상 스트리밍 데이터가 입력 소스로 전달되어 영상과 실제 응답이 일치할 것입니다. 실제 동작 예시는 데모 영상 : 실제 서비스 데모영상 (클릭) 을 확인해 주시기 바랍니다.
